# Ad Analytics Dashboard

A simple project for analyzing and displaying numbers coming from advertisements in a clear and structured way.

## Project Idea

This page collects ad data and analyzes it to provide a quick overview of campaign performance.

## Features

- Display total ads
- Track performance (Clicks / Views / Conversions)
- Simple statistical overview
- Clean and easy-to-use interface

## Goal

To simplify understanding ad results and provide quick insights for better decision-making.

## Technologies Used

- HTML
- CSS
- JavaScript

## Note

The project can be extended with more advanced dashboards and data visualization features.
